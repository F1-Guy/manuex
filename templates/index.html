ManueX

ManueX is the first ever smart irrigator system for green facades that uses both real-time
weather API and sensor data. Thanks to the access to that data, it is able to, with great
precision, predict the need of the green facade to be watered with an algorithm. In the end, ManueX
also connects to a microcontroller and waters the system. On the left there is a menu that links
towards functions like checking the current weather forecast, observing the humidity sensor data,
setting the watering conditions and, last but not least, an override button that waters the plant
instantly.
